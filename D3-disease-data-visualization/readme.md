# Climate Change Burden of Disease Visualization

In this project, bind data to div elements and create a simple bar chart to display the climate change burden of disease by World Health Organization region. Create a legend list that will highlight the corresponding bar when user hover over the corresponding legend item and vice versa. 

Find the raw data describing [“climate change attributable deaths”](http://apps.who.int/gho/data/node.main.132?lang=en) and the different regions at the World Health Organization [repository](http://apps.who.int/gho/data/node.wrapper.imr?x-id=2391).
